# 📖 Про проєкт ("About")

**Navchai+** — це сучасна система керування навчанням ("Learning Management System", LMS), яка розробляється як відкрита ("Open Source") платформа для організації корпоративного навчання, перевірки знань та сертифікації персоналу.

Проєкт створюється з використанням сучасних технологій на базі **Django** та орієнтований на організації, які прагнуть самостійно керувати процесом навчання без залежності від комерційних хмарних сервісів.

Navchai+ дозволяє централізовано адмініструвати курси, уроки, тестування, співробітників та результати навчання в єдиному середовищі.

Архітектура системи побудована за модульним принципом, що дозволяє легко масштабувати функціональність та адаптувати платформу під потреби конкретної організації.

# 🚀 Встановлення ("Installation")

### 1. Клонування репозиторію

```bash
git clone https://github.com/spoil7/NavchaiPlus.git
```

```bash
cd NavchaiPlus
```

---

### 2. Створення віртуального середовища ("Virtual Environment")

Windows

```bash
python -m venv venv
```

Linux / macOS

```bash
python3 -m venv venv
```

---

### 3. Активація віртуального середовища

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

---

### 4. Встановлення залежностей ("Dependencies")

```bash
pip install -r requirements.txt
```

---

### 5. Застосування міграцій

```bash
python manage.py migrate
```

---

### 6. Створення адміністратора ("Superuser")

```bash
python manage.py createsuperuser
```

---

### 7. Запуск сервера

```bash
python manage.py runserver
```

Після запуску відкрийте браузер:

```
http://127.0.0.1:8000/
```

Адміністративна панель:

```
http://127.0.0.1:8000/admin/
```
