# 🚀 Про проєкт

**Navchai+** — сучасна платформа цифрового навчання нового покоління, розроблена для організацій, комунальних підприємств, медичних закладів, навчальних установ та інших державних і приватних структур.

Основна ідея проєкту — створити єдину систему, яка дозволяє:

- проводити навчання працівників;
- організовувати тестування;
- автоматично видавати сертифікати;
- аналізувати результати навчання;
- контролювати рівень знань персоналу;
- працювати як локально, так і в хмарі.

---

# 🌍 Гібридна архітектура

Navchai+ створюється як **Hybrid Platform**.

Один код підтримує два режими роботи.

| Режим | Опис |
|-------|------|
| 🏢 Standalone | Локальна установка для однієї організації |
| ☁ Cloud | Хмарна платформа для багатьох організацій |

Це дозволяє використовувати систему як у невеликій організації , так і для сотень організацій одночасно.
