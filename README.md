# 📖 Про проєкт

**Navchai+** — сучасна українська платформа цифрового навчання, яка створюється для автоматизації процесів навчання, перевірки знань та сертифікації працівників.

Проєкт розробляється як **open-source рішення**, яке зможуть використовувати:

- лікарні;
- державні установи;
- навчальні заклади;
- комунальні підприємства;
- приватні компанії;
- будь-які організації, що проводять навчання персоналу.

Головна мета проєкту — створити сучасну українську альтернативу дорогим закордонним LMS-системам, яка буде простою, швидкою, безпечною та доступною.

# 🚀 Запуск проєкту

```bash
git clone https://github.com/spoil7/NavchaiPlus.git

cd NavchaiPlus

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
