# Navchai+

Універсальна платформа навчання, тестування та сертифікації працівників комунальних підприємств та установ України.

## Поточна версія

v0.1 (Prototype)

## Технології

- Python
- Django
- HTML5
- CSS3
- JavaScript

## Архітектура

Платформа підтримує два режими:

- Standalone (локальна установка)
- Cloud (хмарна версія)
